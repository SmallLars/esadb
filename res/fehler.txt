- Disziplin

id > 0
schusszahl > 0
schusszeit > 0
stellungen > 0       1
probeschüsse >= 0    0
probezeit >= 0       0
serienlänge >= 0     5

Die Anzahl der Schüsse muss größer 0 sein.
Die Anzahl Zeit für die Wertungsschüsse muss größer 0 sein.
Die Anzahl der Stellungen muss größer 0 sein.
Die Anzahl der Probeschüsse muss größer/gleich 0 sein.
Die Wert für die Probezeit muss größer/gleich -1 sein.
Die Anzahl der Schüsse pro Serie größer 0 sein.
